## Acknowledgements
The deep learning part of this project builds upon the official FT-Transformer implementation
(Gorishniy et al., NeurIPS 2021). The original model architecture and core
components were used as a reference, while the data loading and experimental
pipeline were adapted to fit our study design.

ChatGPT (OpenAI) was used to assist with code comments, documentation, and minor
code refactoring for improved clarity and readability.
